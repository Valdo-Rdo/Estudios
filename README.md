# 👥 Recursos Pedagógicos PRO - Proyecto Colaborativo 2026

¡Bienvenidos a nuestra plataforma grupal! Este repositorio es un ecosistema digital diseñado como un **Proyecto Escolar Colaborativo**, destinado a ser una fuente de consulta permanente para nuestro grupo. El objetivo es que todos podamos acceder a herramientas pedagógicas y técnicas desde cualquier lugar y dispositivo durante el desarrollo de nuestra carrera.

## 🚀 Propósito del Proyecto
Fomentar el aprendizaje cooperativo mediante un banco de recursos dinámico que integra la pedagogía con soluciones de sistemas, permitiendo que las herramientas de estudio estén siempre al alcance de un clic.

---

## 🛠️ Módulos de la Plataforma

### 📖 1. Guía: Diseño de Materiales Educativos
Compendio interactivo sobre paradigmas educativos, modelos instruccionales y el Diseño Universal para el Aprendizaje (DUA). Incluye alertas de "Reactivos de Examen" para facilitar el estudio técnico.

### ⏳ 2. Línea del Tiempo: Política Educativa
Herramienta visual que permite explorar cronológicamente la evolución de los planes y programas educativos en México (1920 - 2018), analizando reformas y contextos históricos.

### 🎯 3. Simulador de Examen (Próximamente)
Módulo de autoevaluación diseñado para practicar con reactivos tipo examen sobre Seguridad Escolar, Políticas Públicas y Diseño Curricular.

### 🎬 4. Galería Multimedia IA (Dolls IA)
Repositorio de videoclips educativos generados con Inteligencia Artificial, explorando el uso de avatares digitales como recursos didácticos innovadores.

### 📋 5. Plantillas y Formatos APA 7
Base de datos con estructuras estandarizadas para la redacción de ensayos, reportes de lectura y referencias académicas, optimizando los trabajos del cuatrimestre.

### 🎲 6. Generador de Equipos Aleatorios
Herramienta lógica para la gestión de dinámicas grupales, permitiendo organizar equipos de trabajo de forma rápida, equitativa y azarosa.

---

## 🌍 Acceso Universal
La plataforma ha sido optimizada para ser consultada en cualquier momento:
* **Movilidad:** Acceso total desde smartphones para consultas rápidas en el aula.
* **Colaboración:** Código abierto en GitHub para que el grupo pueda proponer mejoras.
* **Permanencia:** Diseñado para acompañarnos como soporte técnico durante toda la formación docente.

---

## 💻 Tecnologías Aplicadas
* **Diseño:** HTML5, CSS3 y **Tailwind CSS** para una interfaz moderna y limpia.
* **Lógica:** JavaScript (ES6+) y **React.js**.
* **IA:** Generación de contenido multimedia avanzado.
